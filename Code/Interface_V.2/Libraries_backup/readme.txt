Dossier secondaire(Libraries_backup)

Contient : 		Fichier ArduinoJson : Librarie permettant le décodage d'un JSON 
			Fichier EspSoftwareSerial : Librarie permettant d'avoir un moniteur série en utilisant des GPIO différents de ceux proposé par l'ESP32
			Fichier ICM42670P : Librairie permettant la communication avec l'accéléromètre
			Fichier LVGL: Librarie avec la bonne configuration
			Fichier TFT_eSPI: Librarie avec la bonne configuration
			Fichier ui : interface généré de eez studio
			Fichier lv_conf.h : configuration de la livrairies de LVGL

Language: C++
Environnement: Arduino avec board manager de ESP32 par Espressif Systems 2.0.14 (une version supérieur ne fonctionnera pas)
Plateforme: Arduino IDE 2.3.6
